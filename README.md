# üõí Simple E-Commerce Store

This project is a full-stack e-commerce application built for **Code Alpha Task 1**. It features a product catalog, user authentication, and a functional shopping cart using the **Express.js** framework.

## ‚ú® Features

As per the task requirements, the following features are implemented:

* **Product Management:** Dynamic product listings and detailed individual product pages.
* **Shopping Cart:** Users can add/remove items and see a live total.
* **Order Processing:** A simulated checkout flow to handle customer purchases.
* **User Authentication:** Secure registration and login system for customers.
* **Persistent Storage:** A database to store product info, user credentials, and order history.

---

## üõ† Tech Stack

* **Frontend:** HTML5, CSS3 (Bootstrap), and JavaScript.
* **Backend:** Node.js with **Express.js**.
* **Database:** MongoDB (using Mongoose ODM).
* **Authentication:** Session-based auth with `bcrypt` for password hashing.

---

## üèó Architecture

The project follows the **Model-View-Controller (MVC)** design pattern to ensure the code is scalable and organized.

* **Models:** Define the structure of Products, Users, and Orders in the database.
* **Views:** EJS templates for rendering the UI (Home, Product, Cart, Login).
* **Controllers/Routes:** Logic to handle user requests and data manipulation.

## üöÄ Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/simple-ecommerce-store.git
cd simple-ecommerce-store

```


2. **Install Dependencies:**
```bash
npm install

```


3. **Configure Environment:**
Create a `.env` file and add your MongoDB URI:
```env
MONGO_URI=your_mongodb_connection_string
PORT=3000

```


4. **Start the Server:**
```bash
node task1.js

```


*Visit `http://localhost:3000` in your browser.*

## üóÑ Database Schema

The database is structured to handle relational-like data in a NoSQL environment:

| Collection | Key Fields |
| --- | --- |
| **Users** | username, email, password (hashed) |
| **Products** | name, price, description, image, stock |
| **Orders** | userID, items, totalAmount, date |
